# Домашнее задание weather app

Сайт: [weather42.jgroup.top](https://weather42.jgroup.top)

Приложение **Weather42** — это веб-приложение для отображения текущей погоды. Реализовано без использования внешних библиотек и фреймворков, с валидной и понятной структурой кода

---

## Функциональность

Приложение позволяет:

1. **Просматривать погоду по городу**
    - Автодополнение при вводе города с помощью [WeatherAPI](https://www.weatherapi.com/)
    - Валидируется ввод
    - Отображаются:        
        - Иконка с погодой
        - Температура
        - Названия города и страны
        - Скорость ветра
        - Описание погоды (солнечно, дождь и т.д)
        - Местное время
        - Интерактивная карта с местоположением города

2. **Просматривать погоду по координатам**
    - Пользователь вводит широту и долготу
    - Валидируется ввод
    - Отображается та же информация, что и для города, с картой

3. **Добавление нескольких виджетов**
    - Каждый поиск создает новый виджет, который сохраняется в истории на странице, можно пролистывать скроллом

4. **Интерактивная карта**
    - Встроена карта OpenStreetMap для выбранного города или координат
    - Маркер указывает точное местоположение

5. **Дизайн**
    - Был разработан в Figma с нуля
    - Тёмная строгая цветовая схема
    - Небольшой адаптив под мобильные устройства
    - Различные иконки погоды (разные для солнечно/дождь и т.д.)

---

## Технологии

- **HTML / CSS / JavaScript**
- **Figma — для проектирования макета и дизайна интерфейса**
- **Fetch API** + `async/await` для работы с API погоды
- **WeatherAPI** — для получения данных о погоде
- **OpenStreetMap** — для отображения карты

Код соответствует всем условиям:
- Методы не превышают 25 строк
- Переменные и функции имеют осознанные имена
- Код структурирован и разделён по функциональности (ввод, запрос API, отображение, история)

## Использование

1. **По городу**: Начните вводить название города → выберите из подсказки → нажмите "Показать погоду"
2. **По координатам**: Введите широту и долготу → нажмите "Показать погоду"
3. Каждый результат создаёт новый виджет в истории
